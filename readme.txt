Insert wider images on your posts. Images are better if big !

== Description ==
This plugin will allow you to insert a full width image inside your post or page.
The desired image will be a 100% full width image and will occupy the entire space of the main div.

== Installation ==
Installation is super easy:

- Install the plugin
- Activate the plugin through the ‘Plugins’ menu in WordPress
- Place the shortcode [ultrawilderimages image=\"\"  height=\"\"  copyright=\"\" copyright_link=\" copyright_color=\"\"]
- Compile the image url, the image height, the copyright label (if needed), the copyright link and the copyright text color.
